# CTA Patterns

## CTA Formula Taxonomy

| Formula | Structure | Example |
|---------|-----------|---------|
| Action + Benefit | [Verb] + [what they get] | Download the free guide |
| First Person | I want to [benefit] | I want my free report |
| Yes + Benefit | Yes! [outcome I want] | Yes! Show me the shortcut |
| Imperative Verb | Single strong verb | Start / Join / Get / Try |

---

## Verb Selection Guide

| Verb | Best When | Funnel Stage |
|------|-----------|--------------|
| Get | Receiving something tangible | TOFU/MOFU |
| Download | Digital asset (ebook, template, tool) | TOFU |
| Start | Beginning a process or trial | MOFU/BOFU |
| Try | Low-commitment trial | MOFU/BOFU |
| Join | Community or membership | TOFU/MOFU |
| Discover | Curiosity-driven, learning offer | TOFU |
| Build | Creation-oriented outcome | MOFU |
| Book | Scheduling a call or demo | BOFU |
| Watch | Video content | TOFU/MOFU |
| Claim | Scarcity or exclusive offer | BOFU |
| Save | Discount or efficiency offer | MOFU/BOFU |
| Unlock | Gated feature or content | MOFU |
| See | Demo or product tour | MOFU |
| Calculate | ROI or assessment tool | MOFU |
| Compare | Competitive context | MOFU |

## CTAs by Funnel Stage

### TOFU — Awareness CTAs (low friction, no commitment)

| Pattern | Example |
|---------|---------|
| Download [free thing] | Download the free checklist |
| Get the [resource] | Get the 2025 benchmark report |
| Watch the [time] video | Watch the 3-minute overview |
| Read the guide | Read the beginner's guide to [topic] |
| Join [N]+ [audience] | Join 12,000+ marketers |
| Discover how [outcome] | Discover how teams cut churn by 30% |

### MOFU — Consideration CTAs (moderate commitment)

| Pattern | Example |
|---------|---------|
| See [product] in action | See the dashboard in action |
| Try [product] free for [N] days | Try it free for 14 days |
| Get my [personalized thing] | Get my personalized roadmap |
| Calculate my [metric] | Calculate my ROI |
| Compare [options] | Compare plans |
| Start my free [trial/audit] | Start my free audit |

### BOFU — Decision CTAs (high intent, conversion-focused)

| Pattern | Example |
|---------|---------|
| Book a [call/demo] | Book a 20-minute demo |
| Start [product] today | Start Acme today |
| Get started — no credit card | Get started — no credit card required |
| Claim [offer] | Claim your 30% launch discount |
| Talk to [team/expert] | Talk to our team |
| [Outcome] starts here | Faster reporting starts here |

## First-Person CTA Patterns

First-person CTAs consistently outperform imperative-only phrasing.

| Standard CTA | First-Person Version |
|-------------|----------------------|
| Download the guide | I want the guide |
| Start your free trial | Start my free trial |
| Get the report | Send me the report |
| Book a demo | Book my demo |
| Join the community | I'm in |
| Claim your discount | Claim my discount |

## Microcopy Patterns (Trust Signals Near CTA)

Place one trust signal within 1–2 lines of the CTA button.

| Trust Signal Type | Example |
|------------------|---------|
| Risk reversal | No credit card required |
| Social proof | Join 8,400 teams already using it |
| Guarantee | 30-day money-back guarantee |
| Effort reducer | Setup takes under 5 minutes |
| Privacy | We never share your data |
| Cancellation ease | Cancel anytime, no questions |
| Speed | Get access instantly |
| Commitment framing | No long-term contracts |

## CTA Placement Guidance

| Placement | When to Use | Notes |
|-----------|-------------|-------|
| Above fold (hero) | Primary page CTA | Must work without supporting copy; benefit-first |
| After problem block | PAS / pain section | Reader is primed; CTA follows emotional peak |
| After proof block | Post-testimonial | Trust is highest here; second CTA on long pages |
| Inline (mid-content) | Blog posts, email | Contextual CTAs convert better than banners |
| Sticky header/footer | High-traffic content | Low-friction; repeats offer without interrupting |
| Exit intent | Pages with high exit rate | Softer offer; email capture or lead magnet |
| End of email | Every email | One primary CTA per email; mirror subject line promise |

---

## CTA Writing Rules

1. One primary CTA per page section — multiple competing CTAs reduce conversion.
2. Button text must name the outcome, not the action — "Get my report" not "Submit."
3. Match CTA language to the headline promise — if the headline says "free guide," the CTA should reference the guide.
4. High-intent pages (pricing, BOFU) can carry a primary CTA and a secondary softer CTA (e.g., "Book a demo" + "Watch a 3-min overview").
5. Size and contrast matter — CTA button must be the most visually prominent element in its section.
6. Test: button color, button text, and microcopy are the highest-leverage A/B variables.
