# Cross-Plan Research Package

Use this template shape for reusable discovery that informs multiple initiatives and should live in `plans/research/`.

## Purpose

- store pre-plan or cross-initiative research outside any single active plan
- separate reusable discovery from initiative-bound `research/` folders
- make later promotion into canonical docs easier

## Minimum Structure

- one research file or a small related set under `plans/research/`
- problem statement or research question near the top
- sources, assumptions, and notable gaps made explicit
- practical takeaways that another plan can reuse

## Promotion Rule

When the research becomes stable repository knowledge, promote it into:

- `docs/reference/knowledge/`
- `docs/foundation/`
- `docs/guides/`
- `docs/decisions/`

Keep `plans/research/` for reusable discovery, not final canonical policy.
