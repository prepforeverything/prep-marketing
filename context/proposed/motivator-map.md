## How to read this map

**Stage 03 (motivators)** is a per-segment inventory. Each motivator is `{motivator · proof ·
source}` and **evidence-graded**:

- **`validated`** — backed by an external dated source (a DataReportal / VnExpress / IELTS-partner
  figure) or by an **approved** claim (`[[CLM-008]]`).
- **`sourced-internal`** — grounded in an approved `context/` file (positioning, products, market
  policy, personas) but not an external statistic.
- **`hypothesis`** — derived from the Stage-01 consumer-voice phrase bank (a creative hypothesis
  for hook testing, explicitly *not* survey-quoted) or from the Stage-00 structural pattern.
- **`anecdotal`** — plausible learner truth with no attached source → **quarantined**, never
  asserted.

This is a **representative first pass** (~8–12 motivators/segment), not the exhaustive set; it is
**extensible** — add motivators as real comments/DMs/search and the full claims set are validated.

**Stage 04 (force-map)** clusters each segment's motivators by **JTBD force** — *push* (what
pushes the learner away from the status quo), *pull* (what attracts them to the new solution),
*anxiety* (what makes them hesitate / fear the switch), *habit* (the inertia / current behaviour
holding them in place). Per segment a **forces-balance** says what to **amplify** vs **defuse**,
and pairs each force with a **Fogg** friction (the B=MAP lever to reduce) and a **Cialdini**
trigger (the influence principle to deploy). Naming the forces this way — not generic themes —
is the deliverable: it tells the brief stage *which lever the creative pulls*.

**Link-check:** every external source below traces to Stage 01 (`market-research.md` §S1–S5 /
§Sources) or to a read approved `context/` file. No new external URL is introduced at this stage;
where Stage 01 itself quarantined a source (Q1–Q7), this map inherits that quarantine.
