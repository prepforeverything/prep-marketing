# Brand Voice & Terminology

## Voice: Clear, Helpful, Human

We sound like **a dedicated teacher in a 1-on-1 conversation** — direct but respectful,
friendly but not overly familiar.

- **Clear** — say it plainly. Short sentences. No jargon the learner wouldn't use.
- **Helpful** — every line should guide the learner toward their next step or their goal.
- **Human** — warm and encouraging, never robotic, never hype-y or pushy.

## Brand Values

- **Dedication** — we mentor the learner from start to finish.
- **Recognition** — small wins matter; celebrate progress.
- **Helpfulness** — every interaction is an opportunity to guide.

## Voice presets by segment

Same brand voice, tuned per audience (full personas in [`audience-personas.md`](./audience-personas.md)).
Pick the preset that matches who the copy is for. Both obey every rule below: Vietnamese-first, no
unapproved claims, admit trade-offs, one CTA.

### Students — lộ trình & áp lực kỳ thi
- **Register:** peer-mentor — warm, encouraging, second person ("bạn"), a little energetic.
- **Lead with:** the goal band and the deadline ("kịp kỳ thi tháng X"), peer proof, a clear next step.
- **Proof + CTA:** band gains and success stories; CTA = kiểm tra band / tư vấn miễn phí.
- **Avoid:** corporate tone, heavy jargon, fear-mongering.

### Professionals — bận rộn & hiệu quả
- **Register:** respectful adviser — concise, time-aware, second person ("anh/chị"), credible.
- **Lead with:** flexibility (tự học theo lịch), efficiency (AI chấm chữa 24/7), the outcome for
  work/immigration.
- **Proof + CTA:** time-to-result and flexible schedule; CTA = đặt lịch tư vấn / nhắn Zalo.
- **Avoid:** student slang, hype, anything that reads as "easy".

## Language policy

> **Vietnamese-first for all customer-facing copy.** Vietnam is our largest market. Write
> the Vietnamese version as the primary, then localize for other markets per
> `markets/<market>.md`. Keep English for internal structure, labels, and documentation.

## Brand Terms (spelling and usage)

| Term | Rule |
|------|------|
| **Prep** | The brand name. Always capitalized as "Prep". In code, use the `{brand_name}` variable for B2B white-labeling. |
| **PREP** | ALL CAPS only in logos / marketing lockups. Do not use ALL CAPS in body copy. |
| **PrepBee** | The mascot. |
| **Teacher Bee AI** | The AI learning assistant. |
| **Preppy** | Informal learner nickname. **Avoid in UI copy.** |
| **Prep IELTS**, **PrepTalk** | Product names. (Confirm exact spacing with the team.) |

## Terminology Governance

Use the **Preferred Term** consistently. This keeps the product and the marketing aligned.

| Concept | Preferred term | Notes |
|---------|----------------|-------|
| Redo / Retry / Retake | **Retake** (completed tests), **Retry** (everything else) | |
| Exam / Test | **Test** (UI), **Exam** (narrative variation) | |
| Hierarchy | **Program** > **Course** > **Unit** > **Lesson** | "Available Plans" in learner-facing copy |
| See / View | **See** (learner-facing), **View** (system-level) | |
| Continue / Keep | **Continue** (buttons), **Keep** (text only) | |
| Analyze / Check | **Analyze** (AI-powered), **Check** (quick) | |
| Answer / Submission | **Answer** (learner responses), **Submission** (file uploads) | |
| Study Plan / Roadmap | **Study Plan** (learner-facing), **Roadmap** (internal) | |
| Practice / Exercise | **Practice** (mode), **Exercise** (individual task) | Avoid **Activity** |
| Score / Band | **Score** (general), **Band** (IELTS-specific) | Avoid **Mark** (UK), **Grade** |
| Hint / Tip | **Hint** (in-exercise, may cost gems), **Tip** (free advice) | Avoid **Clue** |

> When localizing to Vietnamese, keep these English product terms where they are the
> learner's expected vocabulary (e.g. "Band", "Study Plan"), and translate surrounding copy.
> When in doubt, match how the in-product UI already speaks to the learner.
