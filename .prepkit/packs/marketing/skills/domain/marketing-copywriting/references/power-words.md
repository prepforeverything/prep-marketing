# Power Words

## Power Word Placement Guide by Position

| Position | Goal | Recommended categories |
|----------|------|----------------------|
| Headline | Stop the scroll | Urgency, Exclusivity, Curiosity, Authority |
| Subheadline | Sustain interest | Value, Results, Simplicity |
| CTA button | Drive action | Action, Urgency, Value |
| Body copy | Build trust | Trust, Results, Authority |
| Testimonial pull quote | Amplify proof | Emotion, Results |
| Risk reversal / microcopy | Remove hesitation | Trust, Simplicity |

---

## Urgency

Words that create time pressure or scarcity.

| Word | Usage Note |
|------|-----------|
| Now | Effective in CTAs; pair with a reason |
| Today | Specific to day; avoid if no real deadline |
| Limited | Implies scarcity; must be true |
| Deadline | Use with a real date for credibility |
| Expires | Strong in email subject lines |
| Before [date/event] | Adds specificity to urgency |
| Last chance | Reserve for genuine final offers |
| Running out | Product or seat scarcity |
| Immediately | Fast delivery/access promises |
| Act fast | Pair with a specific reason |

---

## Exclusivity

Words that make the reader feel chosen or privileged.

| Word | Usage Note |
|------|-----------|
| Exclusive | Works in invite or access contexts |
| Private | Implies a closed community or resource |
| Members only | Works for paid or gated content |
| By invitation | High-end or application-based |
| Insider | Works well in email and newsletter hooks |
| VIP | Use carefully; can feel cliche if overused |
| Early access | Strong for launches and waitlists |
| Reserved | Event or capacity contexts |
| Select | Works with audience targeting |
| Founding | Strong for early customer framing |

---

## Value

Words that signal a gain or deal.

| Word | Usage Note |
|------|-----------|
| Free | Most powerful value word; be specific about what's free |
| Bonus | Adds perceived value to offers |
| Save | Quantify where possible ("save 4 hours") |
| Gain | Outcome-oriented; more specific than "get" |
| Discover | Implies new value about to be revealed |
| Unlock | Gated feature or knowledge framing |
| More | Use with specificity ("3 more leads/day") |
| Double | Strong but requires evidence |
| Extra | Additive value; good for bundles |
| Maximize | Works in B2B ROI contexts |

---

## Trust

Words that reduce hesitation and build credibility.

| Word | Usage Note |
|------|-----------|
| Guaranteed | Must be backed by a real guarantee |
| Proven | Always pair with data or social proof |
| Verified | Works with certification or review contexts |
| Trusted | Pair with number ("trusted by 5,000+") |
| Safe | Security, privacy, or financial contexts |
| Secure | Especially effective in checkout or SaaS |
| Certified | Professional services, compliance |
| Transparent | Works in pricing and process explanations |
| Honest | Disarming; effective in email |
| Risk-free | Must be backed by a real refund policy |

---

## Results

Words that project outcomes.

| Word | Usage Note |
|------|-----------|
| Transform | Use with before/after framing |
| Achieve | Goal-oriented; works in B2B |
| Accelerate | Implies speed of results |
| Boost | Vague unless quantified |
| Grow | Generic; always quantify ("grow by 40%") |
| Improve | Softer than transform; good for incremental gains |
| Increase | Pairs well with metrics |
| Generate | Revenue or lead generation contexts |
| Reduce | Cost, time, or effort reduction |
| Eliminate | Strongest version of "reduce"; use for big pain points |

---

## Emotion

Words that create a feeling connection.

| Word | Usage Note |
|------|-----------|
| Finally | Validates frustration; "Finally, a tool that..." |
| Imagine | Opens a desire picture |
| Love | Strong if earned; avoid in cold B2B |
| Afraid | Agitation in PAS; use with care |
| Excited | Works in launch copy |
| Relief | After agitation in PAS structure |
| Proud | Works in results / outcome contexts |
| Frustrated | Names a pain; disarming in headlines |
| Confident | Works in empowerment-oriented copy |
| Tired | Pairs with "of [problem]"; relatable opener |

---

## Curiosity

Words that create an information gap.

| Word | Usage Note |
|------|-----------|
| Secret | Overused; use only if there's a real reveal |
| Hidden | Works in auditing, analysis, discovery contexts |
| Revealed | Strong for uncovering frameworks or data |
| Behind the scenes | Process transparency or culture copy |
| Surprising | Effective in headlines; must deliver |
| Unexpected | Works in results or case study copy |
| Counterintuitive | High engagement; good for contrarian angles |
| The truth about | Strong opener; delivers on skepticism |
| What most [people] miss | Addresses gap in common knowledge |
| Why [assumption] is wrong | Contrarian headline starter |

---

## Authority

Words that signal expertise or status.

| Word | Usage Note |
|------|-----------|
| Expert | Pair with specific credentials or evidence |
| Leading | Pair with category or market rank data |
| Industry-standard | Works in technical or compliance copy |
| Pioneer | Origin or invention story |
| Official | Certification or partnership context |
| Award-winning | Must be verifiable |
| Recognized | Conference, press, or analyst validation |
| Backed by | Investor, partner, or research context |
| Research-based | Always link or cite |
| Pioneered | Stronger "first mover" claim |

---

## Simplicity

Words that lower cognitive effort and friction.

| Word | Usage Note |
|------|-----------|
| Easy | Pair with specifics ("easy 3-step setup") |
| Simple | Works in product and process descriptions |
| Effortless | High-friction pain contexts |
| Instant | Delivery or access speed |
| Seamless | Integration or workflow contexts |
| Plug-and-play | Technical onboarding contexts |
| No-code | Developer-dependency removal |
| Done-for-you | Service or template contexts |
| Automatic | Recurring or trigger-based features |
| One click | Literal action; use only when true |

## Words to Avoid

### Vague / Meaningless

| Word | Why to Avoid |
|------|-------------|
| World-class | No definition; cannot be verified |
| Best-in-class | Same as above |
| Cutting-edge | Overused; reader-tuned-out |
| Innovative | Means nothing without specifics |
| Robust | Corporate filler |
| Scalable | Vague in marketing copy; use in technical docs only |
| Synergy | Legacy corporate speak |
| Leverage | Overused in B2B; replace with "use" or specific action |
| Empower | Vague without evidence; replace with outcome |
| Solutions | "We provide solutions" says nothing |

### Overused Marketing Words

| Word | Better Alternative |
|------|-------------------|
| Amazing | Remarkable, exceptional + specific claim |
| Game-changing | [Specific result] changed how we... |
| Revolutionary | First [category] to [specific thing] |
| Disrupting | Replacing / challenging / replacing the old model |
| Passionate about | Replace with proof of commitment |
| Seamless experience | [Specific friction you removed] |
| Holistic | Comprehensive / end-to-end + specifics |
| Turnkey | Ready to use in [N] minutes |
