## Marketing Pack — Quick Start

### Recommended Starting Flow (non-technical, Vietnamese-first)

1. `/mkt-setup` — one time: checks tools, walks you through your brand/product context, and
   (optionally) connects analytics read-only.
2. `/mkt` — the front door for *any* marketing task; it shows a simple menu and routes you to the
   right command.
3. `/mkt-campaign` — a full, guided campaign end-to-end (the golden path).

### Everyday Commands

| Command | Purpose |
|---|---|
| `/mkt` | Front door — routes any task to the right place |
| `/mkt-campaign` | Guided end-to-end campaign (the golden path) |
| `/mkt-build-landing-page` | Conversion landing page, claims-checked |
| `/mkt-write-blog` | SEO-aligned blog article, Vietnamese-first |
| `/mkt-social-pack` | Platform-native social pack (Facebook / TikTok / Zalo / Instagram) |
| `/mkt-email-sequence` | Email / Zalo lifecycle sequence |
| `/mkt-launch` | Product / course launch plan |
| `/mkt-seo-audit` | SEO / ASO audit, prioritized findings |
| `/mkt-report` | Performance report by funnel / channel |
| `/mkt-generate-asset` | Brand-aligned image / short-video creative |
| `/mkt-research` | Research a market / competitor → PROPOSES context updates (never overwrites) |
| `/mkt-connect` | Connect a tool safely (read-only first) |

### Workflows (the playbooks the commands follow)

| Workflow | When to Use |
|---|---|
| `mkt-campaign-golden` | The default guided campaign (what `/mkt-campaign` runs) |
| `marketing-launch` | Simple campaign launch (audience → channels → approvals → review) |
| `marketing-go-to-market` | Full GTM motion (context → positioning → messaging → channels → launch → measure) |
| `marketing-content-pipeline` | Content lifecycle (strategy → SEO → create → optimize → track) |
| `marketing-conversion-optimization` | CRO cycle (baseline → audit → test → analyze) |
| `marketing-growth-loop` | Growth engineering (retention → loops → amplification → iteration) |
| `marketing-lifecycle-flow` | Retention / lifecycle automation (segment → nurture → reactivate) |
| `marketing-campaign-sprint` | Time-boxed campaign sprint |

### Commands

Use `/mkt` (the front door) and the `/mkt-*` commands — each runs the right skills plus the brand +
claims review under the hood. Full skill list: `.prepkit/packs/marketing/pack.manifest.json`.
