## Message Testing Guidance

### What to test

- Headline vs. headline (value prop framing)
- Proof point type (stat vs. testimonial vs. case study)
- CTA copy (action verb choice)
- Persona-specific variants

### How to run an A/B test on messaging

1. **Isolate one variable**: Change only the headline or only the proof point — not both.
2. **Define success metric before launch**: Click-through rate, sign-up conversion, demo request rate.
3. **Set minimum sample size**: Use a significance calculator; aim for 95% confidence, minimum 500 conversions per variant.
4. **Run to completion**: Do not stop early on positive signals — wait for statistical significance.
5. **Document winners**: Record winning variant, margin of improvement, and date in the proof points table above.
6. **Propagate winners**: Update this framework and push the winning message to all relevant channels within two weeks of a confirmed result.
