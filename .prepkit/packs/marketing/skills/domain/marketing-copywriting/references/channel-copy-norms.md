# Channel Copy Norms

## Content Ratio Guidance

| Mix | Description |
|-----|-------------|
| 80% value | Educational, entertaining, or community-relevant content |
| 20% promotion | Direct product promotion, offers, CTAs |

Apply this ratio across a content calendar, not within a single post.

---

## Email

| Element | Norm | Notes |
|---------|------|-------|
| Subject line length | 30–50 characters | 6–9 words; mobile preview shows ~35 chars |
| Subject line sweet spot | 41 characters | Based on open rate data |
| Preview text | 40–90 characters | Extend the subject line; never duplicate it |
| To field | First name only or first + last | Personalized "to" reduces spam score |
| Preheader | 85–100 characters | Treated as second subject line |
| Body — short format | 50–100 words | Cold outreach, transactional, re-engagement |
| Body — standard | 100–200 words | Nurture, product updates |
| Body — story / newsletter | 300–800 words | Lead magnet, case study, long nurture |
| CTA count | 1 primary | Second CTA allowed if secondary offer is softer |
| Image usage | Text-first is safer | Image-only emails often flagged or blocked |
| Mobile rendering | Single column | 600px max width; 14–16px body text minimum |

### Subject Line Patterns

| Type | Template |
|------|---------|
| Curiosity gap | You won't believe what [X] said about [topic] |
| Benefit-driven | How to [outcome] in [timeframe] |
| Question | Are you making this [topic] mistake? |
| Urgency | [Offer] ends [day/date] |
| Personal | Quick question about [their thing] |
| Number-based | [N] [things] for [audience] |

## LinkedIn

| Element | Norm | Notes |
|---------|------|-------|
| Post character limit | 3,000 characters | ~450–600 words |
| Hook (line 1) | Under 150 characters | This is what shows before "see more" |
| Optimal post length | 1,200–1,600 characters | Highest engagement range |
| Line breaks | Single sentences | White space increases readability |
| Hashtags | 3–5 per post | Place at bottom; more than 5 reduces reach |
| Image posts | Use for data, frameworks, carousels | Native documents outperform external links |
| Link posts | Lower organic reach | Use first comment for links when possible |
| Personal > company | Personal profiles reach more | Company pages have 5–10% of personal reach |
| Carousel / document | 7–12 slides | Slide 1 = hook; last slide = CTA |

### LinkedIn Hook Patterns

| Pattern | Example |
|---------|---------|
| Contrarian opener | Everyone says [X]. They're wrong. |
| Story opener | 3 years ago, I made this mistake. |
| Stat opener | 73% of [audience] never do this. |
| Question opener | What happens when [situation]? |
| Number list opener | 5 things I learned from [event/project]: |

## X / Twitter

| Element | Norm | Notes |
|---------|------|-------|
| Tweet character limit | 280 characters | 240 effective (leave room for replies) |
| Hook tweet | Under 140 characters | Shorter hooks share better |
| Thread structure | 8–15 tweets | Tweet 1 = hook; Tweet 2 = credibility/context |
| Thread closing tweet | Recap + CTA | Summarize the value; add reply/retweet ask |
| Images | 2 or 4 perform better than 1 or 3 | Twitter UI crops single images |
| Links in tweets | Reduce reach in timeline | Add link in first reply for organic threads |
| Optimal tweet length | 71–100 characters | Based on engagement data |

## Paid Ads

### Google Ads

| Element | Limit | Notes |
|---------|-------|-------|
| Headline | 30 characters | Up to 15 headlines per responsive ad |
| Description | 90 characters | Up to 4 descriptions |
| Display URL path | 15 characters each (2 paths) | Keyword-rich |
| RSA headlines shown | 3 at a time | Google selects combinations |

### Meta (Facebook / Instagram) Ads

| Element | Limit | Notes |
|---------|-------|-------|
| Primary text | 125 characters shown (500 max) | First 125 visible before "See more" |
| Headline | 27 characters | Feed and right column |
| Description | 30 characters | Below headline in some placements |
| Image text | Under 20% of image area | More text reduces delivery |
| Video caption | Required for silent scroll | 85% of videos watched without sound |

## Landing Pages

| Element | Norm | Notes |
|---------|------|-------|
| Above fold | Headline + subhead + CTA visible without scrolling | Test on 1280px desktop and iPhone SE |
| Fold structure | Pre-headline → Headline → Subheadline → CTA | Social proof element within 100px of CTA |
| Reading pattern | F-pattern on desktop; linear on mobile | Lead with key info in top-left |
| Headline length | 6–12 words | |
| Subheadline | 1–2 sentences | Expand on the headline; add specificity |
| CTA above fold | 1 primary CTA | Low-friction phrasing at TOFU; direct at BOFU |
| Page word count | 500–2,000 words for most landing pages | Complex offers (enterprise) can run longer |
| Paragraphs | 1–3 sentences | Short blocks improve scan rate |
| Social proof placement | After hero, after features, before final CTA | Three placements on long pages |

## Instagram

| Element | Norm | Notes |
|---------|------|-------|
| Caption length | Under 2,200 characters | First 125 chars visible in feed |
| Hook (first line) | Specific, curiosity-driving | Must create reason to tap "more" |
| Hashtags | 3–10 per post | Research-specific over broad |
| CTA in caption | Direct to bio link or story link | Instagram does not allow clickable links in captions |
| Stories CTA | Swipe-up link or link sticker | Link sticker outperforms swipe-up CTA |
| Carousel engagement | Higher saves/shares than single image | Use for educational content |
| Video captions | Required | Auto-captions on, or manual for accuracy |

### Instagram Caption Hook Patterns

| Pattern | Example |
|---------|---------|
| Bold claim | This is the only metric that matters for growth. |
| Direct question | What does your audience actually want? |
| Number list | 3 things I stopped doing to grow faster: |
| Story opening | Last Tuesday I made a mistake in front of 400 people. |

## TikTok / Reels

| Element | Norm | Notes |
|---------|------|-------|
| Hook window | First 1–3 seconds | State the payoff or pose the question immediately |
| Optimal length | 15–60 seconds | Reels: 15–30s for max reach; TikTok: 21–34s sweet spot |
| Captions | Required | Overlaid text for silent viewers |
| Music | Use trending audio | Boosts algorithmic distribution on TikTok |
| CTA timing | In the middle AND end | Mid-video CTAs outperform end-only |
| Hook structure | Statement + why it matters in 3 seconds | "Here's what nobody tells you about [topic] — I tested it for 6 months." |

### TikTok / Reels Hook Patterns

| Pattern | Example |
|---------|---------|
| POV / situational | POV: You just discovered you don't need a degree to do this |
| Challenge convention | You've been told to post every day. That's why you're burned out. |
| Result-first | I went from 0 to 50k subscribers using only one platform. Here's how. |
| Reaction | I asked 100 marketers this question and the answers were shocking. |
