# Content Gap Analysis Framework

A structured process to find content your audience searches for but you do not rank for. Run quarterly.

---

## Step 1: Competitor Keyword Coverage

Find keywords where competitors rank but you do not.

### Process

| Step | Action | Tool |
|------|--------|------|
| 1 | Identify 3–5 primary organic competitors (not just business competitors — the sites outranking you) | Ahrefs Site Explorer "Organic Competitors" |
| 2 | Run a keyword gap analysis comparing your domain against each competitor | Ahrefs "Content Gap" or Semrush "Keyword Gap" |
| 3 | Filter: competitors rank positions 1–20, you rank 0 (not in top 100) | Gap tool filters |
| 4 | Export and tag each keyword by topic area and funnel stage | Spreadsheet |
| 5 | Score by traffic potential × business relevance | Manual scoring |

|---------|------------------|-------------|---------------|-------------|---------|
| [keyword] | [position] | Not ranking | [volume] | TOFU/MOFU/BOFU | H/M/L |

## Step 2: Topic Cluster Gap Audit

Identify missing spoke pages within your existing topic clusters.

### Process

1. List all active pillar/hub pages and their current cluster pages.
2. For each pillar topic, run a keyword research pass to find subtopics you have not yet covered.
3. Check competitor cluster pages — what spoke content do they have that you do not?
4. Identify subtopics with search volume >100/month that have no existing page on your site.

## Step 3: Funnel Stage Coverage Audit

Ensure content exists at every stage of the buyer journey for each key topic.

| Funnel Stage | Content Types | Coverage Check |
|-------------|--------------|----------------|
| TOFU (Awareness) | Blog posts, how-to guides, explainers, statistics pages | Do you have top-of-funnel content for every core topic? |
| MOFU (Consideration) | Comparison pages, case studies, best-of lists, webinars | Do you have comparison/review content for every key product category? |
| BOFU (Decision) | Pricing page, demo page, free trial page, ROI calculator | Do you have clear conversion pages for each use case or segment? |

## Step 4: SERP Feature Opportunities

Identify rich result opportunities your current content is not capturing.

| SERP Feature | How to Qualify | Action |
|-------------|----------------|--------|
| Featured snippet | Rank positions 2–10 for an informational keyword; SERP shows a snippet | Add a direct answer block at top of page; use H2 phrasing that matches the query |
| People Also Ask (PAA) | Target informational keywords; check PAA boxes manually | Add FAQ section addressing PAA questions; add FAQPage schema |
| Knowledge Panel | Strong brand presence + Organization schema | Confirm Organization schema on homepage; populate Google Business Profile |
| Image pack | Visual topic (products, how-to, comparisons) | Optimize image filenames and alt text; use descriptive, keyword-relevant image captions |
| Video carousel | Topic where video is present in top results | Publish video content; add VideoObject schema; submit video sitemap |

---

## Step 5: Content Freshness Audit

Identify stale content that may be losing ranking due to outdated information.

### Criteria for Flagging

| Condition | Action |
|-----------|--------|
| Page not updated in >12 months and ranks for time-sensitive topics | Schedule refresh |
| Statistics pages with data older than 2 years | Update stats; add new sources |
| Competitor comparison pages with pricing/features last verified >6 months ago | Re-verify all data |
| Tutorial or how-to pages referencing deprecated product versions | Update screenshots and steps |
| Any page where a competitor has published a more recent, more comprehensive version | Expand or refresh |

## Step 6: Prioritization Matrix

Score each gap on three dimensions. Use the 3x3 grid to assign action tier.

### Scoring Dimensions

| Dimension | 1 (Low) | 2 (Medium) | 3 (High) |
|-----------|---------|-----------|---------|
| Traffic potential | <100 monthly searches | 100–1,000 | >1,000 |
| Implementation difficulty | New content, complex research | Update existing page | Add section to existing page |
| Business value | Indirect / awareness only | Supports pipeline | Direct revenue or conversion path |

### Priority Grid

| Score (sum) | Priority | Timeline |
|------------|---------|---------|
| 7–9 | P1 — Immediate | This sprint |
| 5–6 | P2 — Near-term | Next 30–60 days |
| 3–4 | P3 — Backlog | 90-day calendar |

## Quick Reference: Gap Analysis Trigger Events

Run or update this analysis when:

- A competitor publishes a significant new content cluster
- A new product feature ships (check if keyword opportunity exists)
- Organic traffic drops more than 15% quarter-over-quarter
- A core topic cluster has not been refreshed in 6+ months
- You are planning a new content sprint and need to validate priorities
