# Brand Consistency Checklist

Use this checklist during cross-channel audits. Collect one sample from each active channel before scoring.

---

## Messaging Alignment

- [ ] Value proposition is consistent across homepage, ads, and sales collateral — same core claim, adapted phrasing
- [ ] Tagline (if any) appears in its approved form; no unauthorized variations in use
- [ ] Key claims (speed, scale, ROI, accuracy) match across all channels — no channel overstates a figure used conservatively elsewhere
- [ ] Tone is consistent across channels — personality markers present even when register shifts (formal in support, casual on social)
- [ ] Proof points (stats, customer outcomes) are cited with the same numbers and attribution on every channel they appear

## Visual Identity

- [ ] Logo appears in approved lockup — correct color version, minimum clear space respected
- [ ] Color palette limited to brand-approved colors; no off-brand accent colors introduced in campaigns
- [ ] Typography follows brand guidelines — correct typefaces, weights, and hierarchy
- [ ] Image style consistent — photography treatment, illustration style, or icon set matches the brand standard

## Voice Consistency

- [ ] Vocabulary follows the do/don't list — restricted words absent, preferred terms in use
- [ ] Tone matches the channel's position on the tone spectrum (see voice-framework.md)
- [ ] Personality markers present — the copy feels like the brand, not a generic template
- [ ] Category or technical jargon is absent in consumer-facing copy; present only where the audience expects it

## Terminology Standardization

- [ ] Product names spelled and capitalized exactly as defined — no abbreviations, no alternate casings
- [ ] Feature names consistent — the same feature is not called different things on the website vs. in-app vs. in ads
- [ ] Category terms aligned — if the brand uses "workspace" internally, no channel uses "project" to mean the same thing

## Proof Point Accuracy

- [ ] Statistics are current — no outdated figures (check if data is more than 12 months old)
- [ ] Customer quotes are attributed correctly — name, title, company present where approved
- [ ] Claims are verifiable — every superlative ("fastest," "most accurate") has a cited source or is qualified

## CTA Alignment

- [ ] CTA matches the brand promise — the offer reinforces the value prop, not a generic "learn more"
- [ ] Urgency level is appropriate — scarcity and time pressure used only when real, not manufactured
- [ ] Friction level matches funnel stage — bottom-funnel CTAs ask for commitment (demo, trial); top-funnel CTAs ask for low-effort action (read, watch)

---

## Cross-Channel Audit Workflow

1. **Collect samples**: Pull one representative asset from each active channel (homepage, email, paid ad, social post, in-app message, sales deck).
2. **Score against checklist**: For each sample, mark each item pass / fail / not applicable.
3. **Identify gaps**: Note which categories have the most failures and which channels are the outliers.
4. **Prioritize fixes**: Fix high-reach channels first (homepage, primary ad creative). Batch low-reach fixes (email footers, old blog posts).
5. **Assign owners**: Each fix needs one owner and a target date. Do not leave gaps as "team" responsibility.
6. **Re-audit after fixes**: Re-check only the failed items, not the full checklist.
